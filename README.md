# 🎥 Multimodal Feature Extraction & Augmentation Pipeline

[![Documentation Status](https://readthedocs.org/projects/mmedia-forgery-detection/badge/?version=latest)](https://mmedia-forgery-detection.readthedocs.io/en/latest/?badge=latest)

This repository contains a professional pipeline designed for synchronized extraction of facial regions and audio features (MFCC) from video datasets, followed by a classical computer vision data augmentation stage. It is optimized for Deepfake detection and multimodal machine learning research.

## 📚 Documentation

The complete technical documentation, including API references for `face2.py`, augmentation logic, and evaluation metrics (ROC, AUC, and EER), is available online:

👉 **[Access the Documentation Here](https://mmedia-forgery-detection.readthedocs.io/en/latest/)**

---

## 🛠 Project Structure

* **`face2.py`**: The core extraction engine. Synchronizes face crops (using MTCNN) with audio MFCC spectrograms (using torchaudio).
* **`data_augmentation.py`**: Post-processing module that applies visual noise, color space transformations (LAB, HSV, etc.), and blur while preserving integrity.
* **`avaliacao.py`**: Evaluation module focused on performance metrics, generating EER, AUC, and DET/ROC curves for forgery detection.
* **`docs/`**: Technical documentation source files generated by Sphinx.
