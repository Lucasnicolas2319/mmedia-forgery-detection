# 🎥 Multimodal Feature Extraction & Augmentation Pipeline

This repository contains a professional pipeline designed for synchronized extraction of facial regions and audio features (MFCC) from video datasets, followed by a classical computer vision data augmentation stage. It is optimized for Deepfake detection and multimodal machine learning research.

## 🛠 Project Structure

* **`face2.py`**: The core extraction engine. Synchronizes face crops with audio MFCC spectrograms.
* **`data_augmentation.py`**: Post-processing module that applies visual noise and blur to face datasets while preserving audio integrity.
* **`docs/`**: Technical documentation generated by Sphinx using detailed docstrings.
